# Turborepo / Next.js / TypeScript boilerplate

[![CI][lint-badge]][lint-url]
[![CI][tsc-badge]][tsc-url]
[![CI][build-badge]][build-url]

Minimal Turborepo boilerplate.

### Build

To build all apps and packages:

```
npm run build
```

### Develop

To develop all apps and packages:

```
npm run dev
```
